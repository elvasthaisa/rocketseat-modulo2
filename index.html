A Rocketseat é uma plataforma de educação em tecnologia onde é possível estudar de forma prática e dinâmica, de modo a alcançar, no menor tempo possível, seus objetivos como programador. É utlizada uma metodologia orientada à prática, uma stack extremamente enxuta, com apoio de uma comunidade incrível de programadores e muito conteúdo gratuito de qualidade, através de blogs e vídeos no Youtube.

Principais tecnologias utilizadas

  • JavaScript
  • Node
  • React
  • React Native
  • Facebook
  • Github
  • Instagram
